# 💻 Portafolio de Programación en C

¡Bienvenido a mi repositorio de aprendizaje de **C**! 

Este espacio documenta mi trayectoria técnica, desde los conceptos fundamentales hasta el desarrollo de proyectos integradores, demostrando mis habilidades en lógica de programación y estructuración de software.

## 📂 Organización del Repositorio

El contenido está estructurado de manera progresiva para facilitar la navegación:

| Carpeta | Descripción |
| :--- | :--- |
| `01_introduccion_y_basicos` | Sintaxis inicial, tipos de datos y operadores. |
| `02_geometria_y_calculos` | Implementación de algoritmos matemáticos y fórmulas. |
| `03_logica_condicional` | Estructuras de control (`if`, `else`, `switch`). |
| `04_bucles_y_series` | Ciclos (`for`, `while`) y manejo de series numéricas. |
| `05_control_de_flujo_y_validacion` | Validación de datos y manejo de errores. |
| `06_proyectos_parciales` | **Proyectos integradores** (destaca el juego del Ahorcado). |
| `07_ejemplos_utiles` | Casos de uso prácticos y utilidades extras. |

## 🚀 Proyectos Destacados

* **[Juego del Ahorcado](06_proyectos_parciales/juego_ahorcado/):** Un proyecto integrador que aplica estructuras de datos, manejo de cadenas (`string.h`) y lógica de control de flujo.

## 🛠 Tecnologías Utilizadas
* **Lenguaje:** C (ANSI C / C99)
* **Enfoque:** Algoritmia, Estructuras de Datos y Programación Estructurada.
